# Wind Turbine SCADA Project

This repository contains the data, notebooks, outputs, Tableau workbook, and report materials for the Wind Turbine SCADA analysis project.

## Structure

- `data/` - raw and processed datasets
- `notebooks/` - EDA, probability, hypothesis testing, PCA and chi-square analysis
- `outputs/` - generated plots and tables
- `tableau/` - Tableau workbook and dataset export
- `report/` - final report, presentation, and contribution log
- `requirements.txt` - Python dependencies

## Usage

1. Populate `data/raw/wind_turbine_scada.csv` with SCADA data.
2. Run the analysis notebooks in `notebooks/`.
3. Export results into `outputs/` and Tableau files.
